# Pizza Order Console App

A simple console program for ordering pizza.  
The user selects the size, dough, toppings, sauce and cheese. The program calculates the cost and cooking time.

## How to launch

1. Make sure that Python is installed (version 3.13.4+).
2. Download the file `pizza_order.py `.
3. Open the terminal (command prompt) in the file folder.
4. Run the command:
python pizza_order.py

text
