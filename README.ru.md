# Pizza Order Console App

Простая консольная программа для заказа пиццы.  
Пользователь выбирает размер, тесто, начинки, соус и сыр. Программа считает стоимость и время приготовления.

## Как запустить

1. Убедись, что Python установлен(версия 3.13.4+).
2. Скачай файл `pizza_order.py`.
3. Открой терминал (командную строку) в папке с файлом.
4. Запусти командой:
python pizza_order.py

text
